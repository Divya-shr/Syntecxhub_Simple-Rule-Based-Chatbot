# Simple Rule-Based Chatbot

## Overview
A Python-based chatbot that uses pattern matching and rule-based responses to simulate conversation.

## Features
- Greeting Intent
- Help Intent
- Small Talk
- Knowledge Base Queries
- Conversation History Logging
- Interactive Console Interface

## Technologies Used
- Python
- JSON
- VS Code
